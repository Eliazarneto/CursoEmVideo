Resumo do modulo 4

iframe é um frame in line

iframe cria uma janela, que mostra o conteúdo do site, dentro da linha a qual o texto está.

Voce pode adicionar um link, entre um iframe e outro, caso o navegador nao suporte, ele vai direcionar você para o site. assim como pode escrever um texto explicando que o navegador não suporta.

da pra mudar o tamanho tbm, com o height e width, usando css ou fazendo a mudança após o frameborder.

scolling cria a possibilidade de deixar o iframe rolar, ou não,
os parametros são, auto, yes ou no.

o frameborder so aceita 0 e 1. com ou sem borda, e dentro de style voce pode criar a border e personalizar como quiser.

Colocar iframe com conteúdo local
para colocar o iframe voce so precisar adicionar a página normalmente, e colocar dentro do src.

dentro do style voce pode centralizar a tabela, normalmente, colocando display block.

é bom sempre adicionar uma mensagem de erro para o iframe 

****cap 22 aula 04 ***
navegaçao no iframe

se voce criar um iframe vazio, colocar a tag nome, voce pode criar links dentro da página que vai interagir abrindo a página dentro do iframe. 

voce precisa criar um link, e adicionar um target que vai apontar para o iframe. lembrar de colocar o name.

ao colocar um iframe, sem src, ele vai abrir transparente, sem exibir conteudo. 
porém, voce pode apagar o src e colocar srcdoc e dentro dele escrever o código do que voce quer que apareça

****cap 22 aula 05 ***
nunca montar um site com muito iframe, ou conteudo muito importante, pois o google bot pode não entrar dentro do iframe e não faça a identificação do conteúdo.
